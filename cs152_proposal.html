Can Computers Read Bad Handwriting?

Think of this as a more experimental machine learning project. While there is some support already for handwriting to text, there is no real test for how useful this tool is for messier handwriting. I am proposing a sort of "yes" or "no" for whether or not some pencil marking is actual English handwriting, or if it simply scribbles. We would probably have to create our own dataset, but we may also need to survey others since many people have a distinct form of handwriting. The goal would be to train a neural network to determine whether some text is handwriting, or if it is simply just scribbles. If all goes well, the main extension is to run this on even messier data.
